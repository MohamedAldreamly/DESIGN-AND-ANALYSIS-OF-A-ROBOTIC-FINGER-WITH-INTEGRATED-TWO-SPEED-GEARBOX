# Robotic Finger with Two-Speed Gearbox

## Overview
This project presents the design and analysis of a robotic finger actuated by a compact two-speed gearbox. The system combines mechanical simplicity with efficient motion transmission by using direct coupling instead of traditional tendon-driven mechanisms.

The finger motion is achieved using a dual four-bar linkage system that ensures coordinated movement of the phalanges.

---

## Key Features
- Two-speed gearbox (Speed Mode & Force Mode)
- Direct mechanical coupling (no tendons required)
- Coordinated motion using four-bar linkage mechanisms
- Compact and efficient mechanical design
- Designed for educational and experimental applications

---

## System Description
The system consists of two main subsystems:

### 1. Gearbox
- Multi-stage spur gear system
- Two operating modes:
  - Speed Mode → fast motion
  - Force Mode → high torque output
- Dog clutch mechanism for switching between modes

### 2. Finger Mechanism
- Three phalanges connected by revolute joints
- Motion coordination achieved using two coupled four-bar linkages
- Direct connection between output shaft and first phalanx

---

## Design Process
The project was developed through the following steps:

1. Kinematic analysis of finger motion  
2. Four-bar linkage synthesis  
3. Torque estimation and gearbox design  
4. Gear selection and force calculations  
5. Shaft design (bending + torsion)  
6. CAD modeling using SolidWorks  
7. Motion simulation  
8. Structural validation using FEA  

---

## Analysis & Validation
- Analytical calculations for gears and shafts  
- Finite Element Analysis (FEA) for:
  - Output shaft  
  - Gear teeth  
  - Dog clutch  
- Validation of stress distribution and critical regions  

---

## Key Results
- Output torque achieved: ≈ 1.94 N·m  
- Smooth and coordinated finger motion  
- Good agreement between analytical and FEA results  
- Critical stress concentrations identified and analyzed  

---

## Limitations
- Stress concentration at shaft shoulders  
- Gear tooth root is the most critical region  
- Dog clutch engagement may introduce shock loads  
- No fatigue life analysis (future work)  

---

## Future Improvements
- Fatigue analysis under cyclic loading  
- Optimization of shaft geometry (fillet radius)  
- Improved clutch engagement mechanism  
- Prototype manufacturing and testing
